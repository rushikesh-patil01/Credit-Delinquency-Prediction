# Credit Delinquency Prediction

This project uses Machine Learning techniques to predict potential credit delinquency risks using customer financial data.

## Technologies Used

* Python
* Pandas
* NumPy
* Seaborn
* Matplotlib
* Scikit-learn

## Machine Learning Algorithm

* Random Forest Classifier

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Encoding
5. Train-Test Split
6. Model Training
7. Prediction & Evaluation

## Model Accuracy

Achieved approximately 93% prediction accuracy
